## Phase A: 部品調達

### チェックリスト

- [ ] **モーター発注** — Nidec 24H055M020 ×3（+予備1）
  - 購入先: AliExpress or Amazon.co.jp
  - 到着目安: AliExpress 2-3週間 / Amazon 数日
- [ ] **マイコン発注** — Raspberry Pi Pico W ×1
  - 購入先: 秋月電子 or スイッチサイエンス
- [ ] **IMUセンサ発注** — MPU6050モジュール (GY-521) ×1
- [ ] **電源部品** — 7805レギュレータ、コンデンサ(0.1µF, 33µF)、抵抗(1.5kΩ, 470Ω)
- [ ] **バッテリー** — LiPo 3S 11.1V 500-800mAh
- [ ] **バッテリー充電器** — 3S対応バランス充電器
- [ ] **コネクタ類** — ピンヘッダ、ピンソケット、配線材
- [ ] **ネジ類** — M3ボルト各種、M3ナット、皿小ねじ、スペーサー
- [ ] **シャフトハブ** — 6mm径用 金属製 ×3
- [ ] **電源スイッチ**
- [ ] *(オプション)* INA219モジュール
- [ ] *(オプション)* Bluetoothモジュール HC-05/HC-06

### 並行作業（待ち時間を有効活用）
- モーター到着待ちの間に Phase B（PCB発注）を開始
- ファームウェアの理解・カスタマイズを進める

---

## Phase B: PCB作成

### チェックリスト

- [ ] **ガーバーデータ確認** — `hardware/gerber/` のデータをビューアで確認
- [ ] **PCB発注** — JLCPCB or PCBGOGO にアップロード・発注
  - ガーバーデータ: `D:\3軸姿勢制御\2号機_Pico版\回路図、ガーバーデータ\` を使用
  - 到着目安: 1-2週間
- [ ] **PCB到着・外観確認**
- [ ] **はんだ付け**
  - ピンヘッダ/ピンソケット取付け
  - 7805レギュレータ、コンデンサ、抵抗の実装
  - 通電テスト（テスターで5V出力確認）

---

## Phase C: メカ組立

### 前提条件
- 3Dプリントパーツ印刷済み ✅

### チェックリスト

- [ ] **リアクションホイール作成**
  - wheel_x3.stl ×3 にシャフトハブを取付け
  - 外周にボルト＆ナットで重量バランス調整
- [ ] **モーターブラケット作成**
  - BRKT_motor1_x3 + BRKT_motor2_x12 を組立て
  - モーターをブラケットに固定（低頭ねじ推奨）
- [ ] **モーター側フレーム組立て** (3面)
  - FRAME_motor_x3 にモーターブラケットをビス止め
  - CORNER_CUBEx8 をフレーム角に取付け
- [ ] **基板側フレーム組立て** (3面)
  - FRAME_bord_x3 を組立て
  - BRKT_MPU6050_x1 にMPU6050を固定
  - BRKT_PCBbord_x1 に基板を固定
- [ ] **全体組立て**
  - モーター側3面 + 基板側3面 を合体
  - リアクションホイールをモーターに取付け

---

## Phase D: 電気接続

### チェックリスト

- [ ] **モーター配線**
  - 3モーター → 基板の8ピンコネクタに接続
  - 配線色とピンの対応を確認（逆接注意！）
- [ ] **センサー接続** — MPU6050 → I2C (SDA/SCL)
- [ ] **バッテリー接続** — LiPo → 基板電源
- [ ] **通電テスト**
  - 5V出力確認
  - I2Cスキャンでセンサー応答確認（0x68）
  - モーター単体回転テスト

### チェックリスト

## Phase F: 1軸テスト（辺倒立）

### チェックリスト

- [ ] **1軸モード** — コードの `vertical == 1` 状態で1モーターのみ制御
- [ ] **辺（稜線）の上に立てて** Roll軸のみの倒立テスト
- [ ] **PIDゲイン調整** — Processing GUI で Kp/Ki/Kd を調整
- [ ] **安定倒立の達成**（数秒以上）

---

## Phase G: 3軸点倒立 & チューニング

### チェックリスト

- [ ] **3軸モード** — コードの `vertical == 2` 状態で3モーター制御
- [ ] **角（頂点）の上に立てて** 点倒立テスト
- [ ] **PIDゲイン調整** — Roll/Pitch同時に調整
- [ ] **Yaw軸制御追加** — Kz パラメータ調整
- [ ] **安定点倒立の達成** 🎉
- [ ] **重量バランス微調整** — ホイール外周のボルトで調整

---

## 📝 注意事項

- **LiPoバッテリーの取扱い注意** — 過放電・過充電・物理的損傷に注意
- **モーター電源の逆接続厳禁** — Nidec 24H の内蔵ドライバが破損する
- **広い平地で作業** — 制御不安定時にキューブが飛び跳ねる可能性あり
- **最初は手で支えながら** — いきなり離さず、制御の挙動を確認してから
